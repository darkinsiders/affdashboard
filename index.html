Everything you need to know to run and update smartlinks
① What is a Smartlink?

A smartlink is a single URL you give to traffic sources (Facebook, TikTok, media buyers). When a visitor opens it, the page instantly detects their country via IP and redirects them to the correct casino affiliate link — all in under 1 second. Each funnel (Dark, Slotio, Scanio, Spin, Shadow) has its own smartlink file.

② How the Redirect Works
1Visitor opens the smartlink URL hosted on GitHub Pages
2The page calls ipapi.co to detect their country code (e.g. DE, SI)
3Their IP is hashed to a number — this maps them to a casino based on your traffic weights (e.g. 70% IVYBET / 30% ROLLETTO). The same IP always gets the same casino.
4If their country is not covered, they go to the Fallback URL you set
5If ipapi.co fails, it automatically retries with freeipapi.com as backup
③ Changing Which Casinos a Funnel Uses
1. Click the funnel tab (e.g. Dark) → scroll down to Smartlinks → Config tab
2. Under Assigned Deals — use the + Add deal dropdown to add a casino, or × to remove one
3. Use ↑ ↓ arrows to change priority order
4. After changing — re-download the smartlink (routes update automatically via Firebase — no re-upload needed)
④ Pushing One Casino More Than Another (Traffic Weights)
Each assigned casino has a % weight input. Total must equal 100%.
Example: IVYBET 70% + ROLLETTO 30% = visitors in shared GEOs get IVYBET 70% of the time, ROLLETTO 30% of the time — and it stays consistent per IP.
Click Auto-balance to split evenly. Weight changes push to Firebase automatically — live smartlinks update instantly.
⑤ Setting the Fallback URL
The fallback is where visitors go if their country is not covered by any assigned casino. Click ← Use [CASINO] (X%) to auto-set it to your top-weighted casino, or paste any URL manually. Always set a fallback — leaving it blank causes a redirect loop.
⑥ Hosting on GitHub Pages (Getting the Live URL)
1. Download the smartlink file (e.g. dark-smartlink.html)
2. Go to your GitHub repo for that funnel → click the file → edit/replace it → commit
3. The smartlink URL is: https://darkinsiders.github.io/dark-/
4. Paste this URL into the Hosted URL field in the dashboard so it stays saved
5. Routes update automatically via Firebase — you only need to re-upload if the smartlink file itself changes (rare)
⑦ Adding a New Deal (Casino)
1. Click + Add Deal in the toolbar → fill in name, affiliate URL, GEOs, and deal terms
2. The GEOs field accepts country codes separated by spaces (e.g. DE AT BE NL) or country names
3. After adding, go to the relevant funnel → Config → assign the new deal
